# DiamondEdge Stat Analyzer

Advanced Sports Analytics Platform with AI/ML.

## Features
- AI-powered player analysis and predictions
- Real-time betting alerts and notifications
- Advanced statistics and visualizations
- Multi-sport support (MLB, NBA, NFL)
- Custom watchlist and tracking

## Tech Stack
- Flutter/Dart (Cross-platform)
- TensorFlow Lite (AI/ML)
- SQLite (Local storage)

## Installation
1. Download from releases
2. Or clone and build from source
